# /setup

Creates `.specorator/catalog-config.md` with user-chosen settings.

1. Ask the user for:
   - `reportFolder` (default: "Reports")
   - `mocFolder` (default: "Maps of Content")
   - `auditDefaults.includeOrphans` (default: true)
   - `auditDefaults.includeBrokenLinks` (default: true)
   - `auditDefaults.maxDepth` (default: 2)
2. Check if `.specorator/catalog-config.md` already exists with `mcp__specorator-obsidian-mcp__vault_read`.
   If it does, show current values and confirm before overwriting.
3. Write the config note with `mcp__specorator-obsidian-mcp__vault_write` to `.specorator/catalog-config.md`:

```
---
reportFolder: "{value}"
mocFolder: "{value}"
auditDefaults:
  includeOrphans: {value}
  includeBrokenLinks: {value}
  maxDepth: {value}
version: 0.1.0
---
# Specorator Catalog Configuration
This note is generated by /setup. Edit values above; assets read this frontmatter.
```

All fields are optional for catalog consumers — assets fall back to documented defaults when absent.
